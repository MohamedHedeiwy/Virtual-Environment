#Start Django projects
Run this command: django-admin startproject FYP --> name of project
 
#Run server
You will notice after you created the django project there is a manage script already created for you, this is used to run the server
Run this command inside of your created django project which in this case its FYP: python manage.py runserver

After you run the command compy the URL path provided into your web browser
